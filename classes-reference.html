Margins

Apply margins using .m{side}-{size} classes. Sides: t (top), b (bottom), l (left), r (right). Standard sizes: 0, xxs, xs, s, r, m, l. Custom: .mt-1 through .mt-100, .mb-1 through .mb-100, etc. (in rem).

Sample Element
Class: .mt-0

Padding

Apply padding using .p{side}-{size} classes. Sides: v (vertical/top+bottom), h (horizontal/left+right). Standard sizes: xs, s, r, m, l, xl, hero. Custom: .ph-1 through .ph-100, .pv-1 through .pv-100 (in pixels, converted to rem).
